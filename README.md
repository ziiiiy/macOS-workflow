# macOS-workflow
Simple tools on macOS built with Automator.

Only tested on macOS 10.15.

Without robustness, and as long as them work.

## Video to .mp4

Need FFmpeg installed.

```bash
brew install ffmpeg
```

将本地的视频文件转为通用的 .mp4 格式。使用默认转码规格，输出在源文件夹。
